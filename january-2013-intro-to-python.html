Intro to Python Workshop

Sat 19 January 2013 by David Fischer

We had over 30 people attend our Intro to Python workshop today. It was a full day of learning Python with food and drinks provided. Thanks to the Python Software Foundation for sponsoring the workshop and the Ansir Innovation Center for hosting. If you have feedback you would like to send about the event or you would like to host or sponsor an event, contact us at sandiegopython@gmail.com.

Next month, we will be having Django Day which is being sponsored by Cuker Interactive and in April we will be having a Python for Data Analysis workshop sponsored by Qualcomm's Open Source Portal group. Sign up before they fill up!
